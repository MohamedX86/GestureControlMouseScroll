# Gesture Control Mouse Scroll

A computer vision project that enables controlling mouse scrolling using hand gestures captured via webcam.

## 🚀 Features
- Hand gesture detection using Mediapipe
- Real-time webcam processing
- Mouse scroll control based on finger movement

## 🛠️ Technologies Used
- Python
- OpenCV
- Mediapipe
- PyAutoGUI
